# 摘要

* 使用 Rust 重写 Zircon 内核，并注意：

  1. 利用 Rust 的**零成本抽象、内存安全、并发安全**等特性；
  2. 利用 Rust 的**异步 async 编程机制**

  如何在裸机环境下做好这些是一个挑战

# 1. 背景与相关工作

* Zircon 是一个全新的微内核，值得借鉴的方面有：
  * 系统调用的设计
  * 系统调用层的 VDSO 设计
  * 相关的用户态驱动机制
  * 为不同硬件架构(x64, ARM64 的主流 PC 以及若干开发板)提供支持
* 参考 Zircon 的设计文档，用 Rust 重新实现它的内核对象，达到能跑起来 Fuchsia 用户程序的效果
